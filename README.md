# Adaptive Cyber Defense Dashboard

A modern Reinforcement Learning (RL) dashboard for DQN-based cyber defense training and monitoring.

## 🚀 Getting Started

Follow these steps to set up the project locally:

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**:
   ```bash
   git clone <your-repo-url>
   ```
2. **Navigate to the project folder**:
   ```bash
   cd adaptive-cyber-defense-dashboard
   ```
3. **Install dependencies**:
   ```bash
   npm install
   ```

### Running the App

1. **Start the development server**:
   ```bash
   npm run dev
   ```
2. **Open your browser** to `http://localhost:3000` (or the port shown in your terminal).

### Environment Variables

Create a `.env` file in the root directory and add your API keys:
```env
GEMINI_API_KEY=your_api_key_here
```

## 🛠️ Tech Stack

- **Framework**: React 19 + Vite
- **Styling**: Tailwind CSS 4
- **Icons**: Lucide React
- **Charts**: Recharts
- **Animations**: Motion

## 🛡️ Features

- **Cyber-Command Center**: Real-time overview of environment states and attack vectors.
- **Mission Control**: Live training simulation with reward curves and resource monitoring.
- **Post-Mission Analytics**: Detailed performance breakdown and historical session tracking.
